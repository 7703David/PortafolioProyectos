# Portafolio de Proyectos

Portafolio de Ingeniería de Sistemas que reúne proyectos académicos y prácticos desarrollados en **Java, C y Python**.  
Incluye implementación de estructuras de datos, programación orientada a objetos, desarrollo de interfaces gráficas y fundamentos de análisis de datos, evidenciando organización, buenas prácticas y crecimiento continuo en desarrollo de software.

---

## Java – PortafolioProyectosJava

En esta carpeta se encuentran proyectos desarrollados en Java, principalmente enfocados en:

- Programación Orientada a Objetos (POO): herencia, encapsulamiento y modularidad.
- Ejercicios de práctica y entrenamiento.
- Aplicaciones con interfaz gráfica (Swing).
- Implementación de estructuras de datos básicas.

Se incluye el proyecto **Cine**, el cual:
- Cuenta con documentación en la carpeta `Docs`.
- Está basado en el ejercicio de la página 418 del libro *Fundamentos de Programación* de Jorge A. Villalobos S.
- Contiene requerimientos funcionales como parte del análisis del sistema.
