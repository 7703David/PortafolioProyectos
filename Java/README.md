# PortafolioProyectosJava

Este apartado del portafolio reúne los proyectos desarrollados en Java durante la formación en Ingeniería de Sistemas. Incluye ejercicios académicos y prácticos enfocados en Programación Orientada a Objetos (POO), estructuras de datos, manejo de colecciones, desarrollo de interfaces gráficas con Swing, validación de datos y lógica de programación.

Los proyectos evidencian la aplicación de conceptos como:
- Encapsulamiento, herencia y polimorfismo.
- Uso de estructuras como `Stack`, `Queue`, `Hashtable` y `Collections`.
- Manejo de excepciones (`try-catch`).
- Desarrollo de aplicaciones gráficas con `JFrame`, `JPanel`, `JOptionPane`.
- Organización de proyectos y buenas prácticas de programación.

Dentro de este portafolio se encuentran ejercicios como:
- Cálculo de área y perímetro de figuras geométricas.
- Simulación de sistemas (Cine, Empresa, Empleados, Automóviles).
- Conversores de unidades.
- Manejo de usuarios y contraseñas.
- Implementaciones de pilas y colas.
- Prácticas de herencia y abstracción.

## TecnologiasComunicacionDistribuida

Este proyecto está orientado a los fundamentos de la programación distribuida y comunicación en red mediante sockets.

Incluye la implementación de un servidor TCP local que permite la conexión de múltiples clientes desde otros equipos o desde la misma máquina.  

Archivo principal:
- `ProgramaServidorTCP.java`

Funcionalidad:
- Crea un servidor TCP que se ejecuta en el equipo local.
- Escucha peticiones de clientes en un puerto específico.
- Permite que diferentes usuarios o clientes se conecten al servidor.
- Sirve como base para comprender la comunicación cliente-servidor, transmisión de datos y manejo de conexiones en red.

Este proyecto demuestra:
- Uso de sockets en Java.
- Conceptos básicos de redes.
- Arquitectura cliente-servidor.
- Comunicación distribuida entre aplicaciones.
- Fundamentos de sistemas distribuidos.

---

Este conjunto de proyectos refleja el progreso en el dominio del lenguaje Java, pasando desde ejercicios básicos de programación hasta aplicaciones con interfaces gráficas y sistemas de comunicación en red, mostrando una evolución técnica sólida y organizada.

## Proyectos de Encriptación

Este portafolio también incluye dos proyectos pequeños enfocados en criptografía básica, orientados a comprender los principios de seguridad de la información y protección de datos:

###  Cifrado César
Proyecto que implementa el algoritmo de cifrado César, uno de los métodos de encriptación más clásicos, basado en el desplazamiento de caracteres dentro del alfabeto.

Permite:
- Encriptar mensajes.
- Desencriptar mensajes.
- Comprender el concepto de cifrado por sustitución.
- Analizar debilidades de los cifrados clásicos.

---

###  Cifrado RSA
Proyecto que implementa el algoritmo RSA, uno de los sistemas de criptografía asimétrica más utilizados.

Incluye:
- Generación de claves pública y privada.
- Encriptación mediante clave pública.
- Desencriptación mediante clave privada.
- Introducción práctica a conceptos como:
  - Números primos
  - Modularidad
  - Seguridad criptográfica
  - Criptografía moderna

---

Estos proyectos fortalecen la base en ciberseguridad y criptografía, mostrando la transición desde algoritmos clásicos hasta esquemas de seguridad usados en sistemas reales.
